# Aura UI - Documentation 🏛️

## Introduction
Thank you for choosing **Aura UI**. This is a premium set of 4 interactive web cards designed with high-end visual effects.

## How to use
1. **Selection Menu:** Start with `index.html` to navigate through all themes.
2. **Customization:** All styles are located in the `.css` files. You can change the `--neon-color` variable to use your own brand colors.
3. **Interactivity:** The `.js` files handle the mouse/touch tracking effects.

## Support
This is a plug-and-play component. For any technical questions, please contact the developer through the marketplace.
